## Music Visualizer in C
This project is a simple program to visualize music data using [Raylib](https://www.raylib.com/) as a graphic API. **Works only on Linux!**

## Demo
https://github.com/user-attachments/assets/65c92bd0-b6b6-4771-9443-6399ab2f79fc


## How to use
- Start the program and **dropped** the music file to enjoy the spectacle!
- Stop and resume by pressing **space bar**
